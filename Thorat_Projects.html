Scratch Project

Project: Scratch Maze Game

Duration: August 29-September 8, 2017

Course: Computer Science and Engineering (CSE)

In this project, the objective was to design a game using the MIT Scratch program My team composed of me, Donna P. and Jacob S. We designed a maze game, which included abundant user interaction, multiple levels and a clear objective to win. These are the requirements for this project.
